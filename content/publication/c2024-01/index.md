abstract: |-
  To address the challenges posed by the growing volume of data and the increasing complexity of user interface navigation in university information systems, this paper introduces a novel data processing framework that integrates large language models (LLMs). The proposed system enhances user interaction through natural language communication, thereby improving both the user experience and adaptability. By leveraging LLM technology, the system streamlines various tasks, including document referencing, database querying, question answering, and data visualization, significantly increasing the efficiency of information retrieval and accessibility for institutional staff. Its high portability and cost-effectiveness render it a promising solution for educational institutions. A pilot implementation has demonstrated the system's capability to comprehend, analyze, and generate accurate responses to user inquiries by leveraging data tables and policy documents.

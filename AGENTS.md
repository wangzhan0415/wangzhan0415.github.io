# Repository Rules

- Do not invent or fabricate any personal biography, academic achievements, publications, projects, awards, education, employment history, research directions, or contact details.
- Any unconfirmed information must be left blank or marked as `待补充` / `To be added`.
- Do not commit private Excel files, application materials, identity documents, student privacy data, internal notes, or other non-public materials to this public repository.
- Run a Hugo build check after every change.
- Do not modify the currently effective Hugo version `0.161.1`.
- Do not delete or disable the GitHub Pages deployment workflow.
